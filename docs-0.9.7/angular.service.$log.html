angular.service.$log

Work In Progress This page is currently being revised. It might be incomplete or contain inaccuracies.

Description

Is simple service for logging. Default implementation writes the message into the browser's console (if present).

This is useful for debugging.

Dependencies

  • $window

Methods

  • $log#log:

    Write a log message

  • $log#warn:

    Write a warning message

  • $log#info:

    Write an information message

  • $log#error:

    Write an error message

Properties

Example

<p>Reload this page with open console, enter text and hit the log button...</p> Message: <input type="text" name="message" value="Hello World!"/> <button ng:click="$log.log(message)">log</button> <button ng:click="$log.warn(message)">warn</button> <button ng:click="$log.info(message)">info</button> <button ng:click="$log.error(message)">error</button>
