angular.filter.currency

Usage

In HTML Template Binding

{{ amount_expression | currency }}

In JavaScript

angular.filter.currency(amount);

Parameters

  • amount(number): Input to filter.

Returns

{string} Formated number.

CSS

ng-format-negative When the value is negative, this css class is applied to the binding making it by default red.

Description

Formats a number as a currency (ie $1,234.56).

<input type="text" name="amount" value="1234.56"/> <br/> {{amount | currency}}
