angular.validator.number

Work in Progress This page is currently being revised. It might be incomplete or contain inaccuracies.

Description

Use number validator to restrict the input to numbers with an optional range. (See integer for whole numbers validator).

Dependencies

    Usage

    In HTML Template Binding

    <input type="text" ng:validate="number[:min][:max]"/>

    In JavaScript

    angular.validator.number(value[, min][, max])

    Parameters

    • value – {string} –

      value to validate

    • min(optional=MIN_INT) – {int} –

      minimum value.

    • max(optional=MAX_INT) – {int} –

      maximum value.

    Example

    Enter number: <input name="n1" ng:validate="number" > <br> Enter number greater than 10: <input name="n2" ng:validate="number:10" > <br> Enter number between 100 and 200: <input name="n3" ng:validate="number:100:200" > <br>
