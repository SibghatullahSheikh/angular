angular.service.$defer

Description

Delegates to $browser.defer, but wraps the fn function into a try/catch block and delegates any exceptions to $exceptionHandler service.

In tests you can use $browser.defer.flush() to flush the queue of deferred functions.

Dependencies

  • $browser

Usage

$defer(fn[, delay]);

Parameters

  • fn – {function()} –

    A function, who's execution should be deferred.

  • delay(optional=0) – {number} –

    of milliseconds to defer the function execution.

Returns

{*} –

DeferId that can be used to cancel the task via $defer.cancel().

Methods

  • cancel(deferId)

    Cancels a defered task identified with deferId.

    Parameters

    • deferId – {*} –

      Token returned by the $defer function.

    Returns

    {boolean} –

    Returns true if the task hasn't executed yet and was successfuly canceled.
