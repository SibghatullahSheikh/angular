angular.module.ng.$compileProvider.directive.ng-mousedown

Description

The ng-mousedown allows you to specify custom behavior on mousedown event.

Usage

as attribute
<ANY ng-mousedown="{expression}">
   ...
</ANY>
as class
<ANY class="ng-mousedown: {expression};">
   ...
</ANY>

Parameters

  • ng-mousedown – {expression} –

    Expression to evaluate upon mousedown. (Event object is available as $event)
