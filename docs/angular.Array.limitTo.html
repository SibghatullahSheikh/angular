angular.Array.limitTo

Work In Progress This page is currently being revised. It might be incomplete or contain inaccuracies.

Description

Creates a new array containing only the first, or last limit number of elements of the source array.

Note: this function is used to augment the Array type in angular expressions. See angular.Array for more info.

Usage

angular.Array.limitTo(array, limit);

Parameters

  • array – {Array} – Source array to be limited.
  • limit – {string|Number} – The length of the returned array. If the number is positive, the first limit items from the source array will be copied, if the number is negative, the last limit items will be copied.

Returns

{Array} New array of length limit.
