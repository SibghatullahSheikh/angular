angular.attrMarkup

Work in Progress This page is currently being revised. It might be incomplete or contain inaccuracies.

Attribute markup extends the angular compiler in a very similar way as angular.markup except that it allows you to modify the state of the attribute text rather then the contest of a node.

angular.attrMarkup('extraClass', function(attrValue, attrName, element){
  if (attrName == 'additional-class') {
    element.addClass(attrValue);
  }
});
