angular.compile

Work in Progress This page is currently being revised. It might be incomplete or contain inaccuracies.

Description

Compiles a piece of HTML or DOM into a scope object.

 var scope1 = angular.compile(window.document);
 scope1.$init();

 var scope2 = angular.compile('<div ng:click="clicked = true">click me</div>');
 scope2.$init();

Usage

angular.compile(element[, parentScope]);

Parameters

  • element – {string|DOMElement} –

    Element to compile.

  • parentScope(optional) – {Object} –

    Scope to become the parent scope of the newly compiled scope.

Returns

{Object} –

Compiled scope object.
